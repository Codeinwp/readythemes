Facebook Open Graph Meta For WordPress help you to add Facebook Open Graph protocol to turn your web pages into the social graph.

== Description ==

Add Facebook Open Graph protocol to turn your web pages into graph objects and enables you to integrate your Web pages into the social graph.

See my article for details and video tutorial [Facebook Open Graph Meta For WordPress](http://www.ivankristianto.com/wordpress-series/facebook-open-graph-meta-for-wordpress/1891/) for details.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `facebook-open-graph-meta-for-wordpress` folder to the `/wp-content/plugins/` directory. or search it from WordPress plugin page.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Facebook Open Graph.
4. Enter your Facebook ID or use the helper to get your Facebook ID.
5. Enable the plugin.
6. Save changes.
